# Training Patches

This directory contains patch files generated by TRAIN-003 (Prompt Patcher) after training debriefs. Each patch file contains structured before/after text edits for competition agent prompts, coordination file templates, or command definitions.

Patch files are named: patch-YYYYMMDD-N.md (date and sequential number).

Patches are reviewed and applied via /apply-training. Applied patches remain here as an audit trail — they are never deleted. If a patch needs to be reverted, a new patch is generated that undoes the changes.
